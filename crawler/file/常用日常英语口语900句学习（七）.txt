一、关于过期问题的常用日常英语口语900句：

292.Those books are overdue.那些书过期了。
293.Are these books overdue？这些书过期了吗?
294.My passport is about to expire.我的护照快要过期了。
295.I'd like to apply for an extension.我想申请延期。
296.Theses books wer due last day.这些书昨天就应该还的。
297.You need to pay a fine for the book.你想要借书需要交罚金。

300.I think this medicine is past-date. You'd better throw it away!我看这药肯定过期了，你最好现在扔掉。

302.If you're going to the library, I might as well go with you and return these books.假如你要去图书馆的话，我倒不如跟你一起去把这些书还了。
